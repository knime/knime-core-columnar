This table backend is currently part of KNIME Labs. It is not yet advised to
use this backend productively.

The columnar table backend was introduced with KNIME Analytics Platform
version 4.2.3. It caches data off-heap and can give noticeable speed-ups over
the Default format. Future versions of KNIME will bring improvements to this
format for increased stability and even greater (order of magnitude) speed-ups.

Please also review the settings in the KNIME preferences
(File → Preferences → KNIME → Table Backend → Columnar Storage (Labs)).
