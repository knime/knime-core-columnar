This table format was introduced with version KNIME Analytics Platform
4.2.2 and is part of KNIME Labs. It uses a more compact data
representation and is, compared to the Default format, more
memory friendly. Using this format can give some noticeable speed-up.
Future versions of KNIME will contain improvements to this format so
that individual nodes will show a significant (order of magnitude) gain
in execution time.

Please also review the global cache settings in the KNIME
preferences (File -> Preferences -> KNIME -> Data Storage).
